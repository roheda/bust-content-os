# v61 · Paleta rápida de colores de marca en BUST It Now

El editor de texto editable ahora lee los colores configurados en el Brand Brain del cliente y los muestra como swatches para aplicar rápidamente al color del texto o al fondo de cada bloque.

Recomendación de carga en Brand Brain: usar valores HEX separados por coma, por ejemplo `#F97316, #FFFFFF, #111827`.

Si no hay HEX, se intenta interpretar nombres comunes como naranja, blanco, negro, azul, verde, rojo, etc.
