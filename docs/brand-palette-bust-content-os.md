# BUST Content OS · Brand palette

Paleta oficial aplicada en la interfaz:

- Verde BUST: `#9EFC7B`
- Arena / fondo cálido: `#EDEAE6`
- Gris oscuro / texto premium: `#343a40`
- Blanco suave / superficies: `#f8f9fa`

## Criterio UX/UI

El verde BUST se usa como acento, estado seleccionado, foco, badges y highlights. Para cumplir contraste y accesibilidad, el texto sobre verde usa `#343a40` en lugar de blanco.

El gris oscuro se usa para acciones primarias, texto principal, íconos y elementos de alto contraste.

El fondo cálido `#EDEAE6` reemplaza el gris frío para que el sistema se sienta más alineado a marca sin perder limpieza tipo Apple.
