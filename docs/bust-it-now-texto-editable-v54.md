# v54 · BUST It Now: modos de texto y capas editables

## Objetivo
Mantener el modo actual de generación con texto IA y agregar un modo profesional para piezas finales donde el sistema genere una base visual sin texto y después monte los mismos bloques como capas editables.

## Modos

### 1. IA escribe el texto
- Es el modo actual.
- Los bloques de texto se mandan al prompt como textos oficiales visibles.
- Sirve para exploración rápida, propuestas completas y referencia de composición.

### 2. Texto editable por BUST It Now
- La IA genera una base visual sin texto legible.
- Los bloques de texto se usan como mapa de composición.
- Después de generar, el usuario puede abrir “Editar texto”.
- El editor crea capas desde los bloques existentes: texto, rol visual, prioridad e instrucciones.

## Editor de texto
Permite:
- mover cada bloque en X/Y;
- ajustar ancho y tamaño;
- cambiar color y fondo;
- alinear texto;
- activar mayúsculas;
- descargar una imagen PNG con texto montado.

## Nota
Esta versión no elimina automáticamente texto pegado por IA si el brief fue generado en modo “IA escribe el texto”. Para piezas finales con copy exacto se recomienda usar “Texto editable por BUST It Now”.
