# Flujo de @menciones y pendientes

- Los comentarios normales ya no dependen de un campo visible de “Dirigir a”.
- Para pedir acción a alguien, se usa `@persona` o `@area`.
- Si al escribir `@` queda una sola coincidencia, Enter o Tab completa la mención automáticamente.
- El panel lateral de dudas en Tareas se elimina para evitar duplicidad con el flotante de Pendientes.
- Las devoluciones automáticas pueden seguir dirigidas internamente a Content/Diseño/Audiovisual para generar pendientes de área.
