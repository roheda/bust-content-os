# BUST Content OS · UI/UX Standards v23

Esta versión aplica una base visual inspirada en iOS/macOS y en principios de usabilidad de Jakob Nielsen.

## Principios aplicados

1. Visibilidad del estado del sistema
- Estados visibles mediante badges, contadores y mensajes claros.
- Pendientes flotantes tipo centro de notificaciones.

2. Consistencia y estándares
- Botones, cards, inputs, tablas y chips usan el mismo sistema de bordes, radios, sombras y colores.
- Navegación agrupada por Operación, IA/Revisión y Dirección.

3. Prevención de errores
- Estados disabled más visibles.
- Focus accesible en inputs y botones.
- Botones destructivos menos agresivos pero claramente diferenciados.

4. Jerarquía y reconocimiento antes que memoria
- Sidebar con iconos y categorías para que el usuario no dependa de recordar dónde está cada cosa.
- Headers más limpios y menos pesados.

5. Accesibilidad
- Contraste más alto en texto principal.
- Focus-visible para navegación por teclado.
- Componentes con tamaños táctiles más cómodos.
- Respetar reduced motion.

6. Estética minimalista
- Se redujo el peso visual de los bloques negros.
- Se aplicaron fondos suaves, glass, blur, sombras ligeras y más aire.

## Estilo base

- Fondo: gris Apple claro.
- Superficies: blanco translúcido con blur suave.
- Acción principal: azul iOS o verde BUST según contexto.
- Texto principal: negro/azul noche de alto contraste.
- Redondeo: tarjetas grandes y botones tipo cápsula.
- Tablas: más aire, encabezados claros, hover suave y ordenamiento visible.
