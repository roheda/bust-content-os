# v69 · Tareas: barra compacta de filtros

## Objetivo
Reducir la altura visual del módulo de Tareas para que el calendario/lista tenga más espacio útil.

## Cambios
- Se compactó el hero de Tareas.
- Se unificaron los tabs de vista y los filtros en una barra compacta.
- La barra usa una sola línea en escritorio y scroll horizontal si no cabe.
- Se acortó "Lista de tareas" a "Lista".
- El aviso de días hábiles se convirtió en badge pequeño.
- Se redujo altura, padding y tamaño de botones/selects.
- Se compactaron las tarjetas KPI de Tareas para ganar espacio vertical.

## Resultado esperado
El usuario debe ver más calendario en pantalla sin tener varios renglones grandes de filtros antes del contenido operativo.
